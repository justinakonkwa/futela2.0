# 📱 Guide d'inscription Commissionnaire

## Comment s'inscrire en tant que commissionnaire ?

Vous avez **2 options** :

---

## Option 1 : Inscription classique (Email/Mot de passe)

### Étapes :

1. **Ouvrir l'app Futela**
2. **Cliquer sur "Créer un compte"**
3. **Sélectionner "Commissionnaire"** (au lieu de "Visiteur")
4. **Remplir vos informations** :
   - Prénom, Nom
   - Email
   - Téléphone (+243...)
   - Mot de passe

5. **Remplir vos informations professionnelles** :
   - Type de pièce d'identité (Carte d'identité, Passeport, etc.)
   - Numéro du document
   - Nom commercial (optionnel)
   - Adresse professionnelle (optionnel)
   - Numéro fiscal / RCCM (optionnel)

6. **Ajouter vos documents** :
   - 📄 Photo de votre pièce d'identité (obligatoire)
   - 🤳 Selfie de vérification (obligatoire)

7. **Cliquer sur "Créer mon compte"**

8. **Votre compte est créé !**
   - Statut : "En attente d'approbation"
   - Un administrateur validera votre demande

---

## Option 2 : Inscription rapide (Google/Apple)

### Étapes :

1. **Ouvrir l'app Futela**
2. **Cliquer sur "Se connecter avec Google" ou "Se connecter avec Apple"**
3. **Authentification Google/Apple**
4. **L'app vous demande de choisir votre rôle**
   - Sélectionner "Commissionnaire"

5. **Compléter votre profil** :
   - Prénom, Nom (pré-remplis)
   - Téléphone (+243...)
   - Type de pièce d'identité
   - Numéro du document
   - Nom commercial (optionnel)
   - Adresse professionnelle (optionnel)
   - Numéro fiscal / RCCM (optionnel)

6. **Ajouter vos documents** :
   - 📄 Photo de votre pièce d'identité (obligatoire)
   - 🤳 Selfie de vérification (obligatoire)

7. **Cliquer sur "Soumettre pour validation"**

8. **Votre profil est complété !**
   - Statut : "En attente d'approbation"
   - Un administrateur validera votre demande

---

## 📋 Documents requis

### Pièce d'identité (obligatoire)
- **Formats acceptés** : Photo (JPEG, PNG, WEBP) ou PDF
- **Taille max** : 8 MB
- **Types acceptés** :
  - Carte d'identité nationale
  - Passeport
  - Permis de conduire
  - Carte d'électeur

### Selfie de vérification (obligatoire)
- **Format** : Photo uniquement (JPEG, PNG, WEBP)
- **Taille max** : 8 MB
- **Conseil** : Prenez une photo claire de vous tenant votre document d'identité

---

## ⏱️ Temps estimé

- **Inscription classique** : ~3 minutes
- **Inscription Google/Apple** : ~2 minutes

---

## ✅ Après l'inscription

### Statut "En attente"
- Votre demande est en cours de validation
- Un administrateur examinera vos documents
- Vous recevrez une notification quand votre compte sera approuvé

### Statut "Approuvé"
- Vous avez accès à toutes les fonctionnalités commissionnaire
- Vous pouvez gérer vos propriétés
- Vous pouvez recevoir des commissions

### Statut "Rejeté"
- Votre demande a été refusée
- Vous pouvez contacter le support pour plus d'informations

## ❓ Questions fréquentes

### Quelle est la différence entre Visiteur et Commissionnaire ?

**Visiteur** :
- Cherche un bien immobilier
- Peut consulter les annonces
- Peut contacter les propriétaires

**Commissionnaire** :
- Professionnel de l'immobilier
- Peut gérer des propriétés
- Peut recevoir des commissions
- Nécessite une validation admin

### Pourquoi mes documents sont-ils obligatoires ?

Pour garantir la sécurité et la confiance sur la plateforme, nous vérifions l'identité de tous les commissionnaires.

### Combien de temps prend la validation ?

La validation prend généralement **24 à 48 heures** ouvrables.

### Puis-je utiliser l'app en attendant la validation ?

Oui ! Vous pouvez utiliser l'app comme un visiteur en attendant l'approbation de votre compte commissionnaire.

### Que faire si ma demande est rejetée ?

Contactez le support via l'app pour comprendre la raison du rejet et soumettre une nouvelle demande avec les corrections nécessaires.
